== Installation ==

= To install plugin =
1.  Upload the entire "woocommerce-gift-wrapper-plus" folder to the "/wp-content/plugins/" directory.
2.  Activate the plugin through the "Plugins" menu in WordPress.
3.  Visit WooCommerce->Settings->Products tab to set your plugin preferences. 
    Additional settings panels are found in each product editor screen.
4.  Please test your gift wrapping options by making mock purchases before going live to make sure it works and looks great!

= To remove plugin: =

1. Deactivate plugin through the 'Plugins' menu in WordPress
2. Delete plugin through the 'Plugins' menu in WordPress
